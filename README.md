# Robert Kabus — Personal Website

A clean, professional personal website built from the original design mockups.

## 🚀 Deploying to GitHub Pages

1. **Create a new GitHub repository** (e.g., `robert-kabus` or `your-username.github.io`)

2. **Upload all files** — make sure these are all in the root of your repo:
   - `index.html`
   - `hero-profile.png`
   - `beliefs-approach.png`
   - `career-experiences.png`
   - `cases-capabilities.png`
   - `roles-responsibilities.png`
   - `guest-lecturer.png`
   - `home-overview.png`

3. **Enable GitHub Pages:**
   - Go to your repo → Settings → Pages
   - Under "Source", select `Deploy from a branch`
   - Choose `main` branch and `/ (root)` folder
   - Click Save

4. Your site will be live at `https://your-username.github.io/your-repo-name/`

## 📝 Customizing

- **LinkedIn link**: Search for `https://linkedin.com` in `index.html` and replace with your actual LinkedIn profile URL.
- **Tagline**: Find "Brand strategist, creative leader..." near the top of `index.html` and update as desired.
- **Content**: All text is in plain HTML — just edit `index.html` directly.

## 🖼 Replacing Images

To use your own photos instead of the screenshots:
- Replace any `.png` file with a new image of the same filename
- Recommended: headshot ~800×1000px, section images ~1400×800px
